# 自查报告 - 2026-07-05

## 检查结论

当前系统已经从“展示型网站”收敛为：

每日推荐 → 隔夜推荐 → 次日复盘 → 胜率统计 → 策略优化。

## 已修正问题

1. 删除旧的首期观察名单文件，避免固定股票被误认为真实推荐。
2. 删除网站里的预约体验和营销入口。
3. 删除假胜率、假回撤、假净值曲线。
4. 模拟推荐和模拟复盘均明确标注为模拟。
5. 首屏真实数据状态改为读取胜率统计，不再永远写死。
6. 模拟提示改为动态状态，未来真实数据写入后不会一直误报模拟。
7. A股和美股每日复盘自动化已要求更新胜率统计文件。
8. 本地预览地址、推荐 JSON、胜率 JSON 已验证可访问。

## 当前模拟数据用途

`latest-picks.json` 和 `latest-review.json` 里的股票和复盘只用于演示页面格式，不是真实推荐，也不能用于实盘决策。

## 仍然缺少的关键能力

1. 可靠实时行情源。
2. A股板块强弱和公告风险源。
3. 美股盘前盘后、财报、新闻和 VIX 风险源。
4. 自动计算买入区、止盈、止损的真实行情模块。
5. 连续多日真实复盘样本。

## 下一步
