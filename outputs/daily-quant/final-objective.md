# 最终目标：持续学习的量化选股助手

## 核心闭环

每天推荐 → 隔夜推荐 → 第二天复盘 → 统计胜率 → 优化策略 → 再推荐。

## 用户真正需要

1. 根据国内外优秀量化交易系统的思路，给出股票推荐。
2. A股和美股都可以覆盖。
3. 单独给出适合隔夜持股的股票。
4. 推荐时必须包含：
   - 当前价格
   - 买入观察区
   - 止盈
   - 止损
   - 卖出/放弃条件
   - 推荐逻辑
   - 风险点
   - 是否适合隔夜
5. 第二天复盘：
   - 开盘涨跌
   - 收盘涨跌
   - 隔夜收益
   - 相对基准表现
   - 是否触发止盈/止损
   - 推荐逻辑是否成立
6. 累计统计：
   - 总胜率
   - 隔夜胜率
   - A股胜率
   - 美股胜率
   - 不同策略方向胜率
7. 每天根据复盘优化策略。

## 不能做的事

- 不能编造当前价格。
- 不能用假胜率、假回撤、假净值曲线冒充真实结果。
- 不能把模拟演示当真实推荐。
- 不能只给ETF而不推荐具体股票。
- 不能只做网站介绍而没有推荐和复盘。

## 网站主要面板

- 今日推荐面板
- 隔夜推荐判断
- 复盘胜率面板
- 策略优化记录

## 真实推荐的前提

必须有可靠行情、成交量、基准表现、板块强弱和事件风险检查。  
数据不足时，只能显示“数据不足”，不能给真实推荐。
